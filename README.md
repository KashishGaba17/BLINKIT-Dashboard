# Blinkit Sales Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard built to analyze sales performance for a quick-commerce business. The dashboard provides a clear view of revenue trends, product performance, outlet characteristics, and customer ratings to support data-driven business decisions.

---

## Tool Used
- Power BI

---

## Key Metrics
- Total Sales  
- Average Sales  
- Number of Items Sold  
- Average Customer Rating  

---

## Dashboard Insights
- Sales distribution across different item types and fat content categories  
- Performance comparison by outlet size, outlet type, and location tier  
- Sales trend analysis by outlet establishment year  
- Identification of top-performing product categories and outlet types  
- Dynamic filtering using slicers for item type, outlet size, and outlet type  

---

## Dashboard Features
- KPI cards for quick business overview  
- Bar, line, and donut charts for comparative analysis  
- Interactive slicers for filtering and drill-down analysis  
- Single-page dashboard designed for clarity and ease of interpretation

## Outcome
The dashboard enables stakeholders to quickly understand key sales drivers, outlet performance, and product trends, helping support informed decisions related to inventory planning and business strategy.
